# Wavy

Wave-based pixel survival game. Made in a one-day school hackathon with Unity.

![preview](Assets/preview.gif)

Play in browser: https://play.unity.com/en/games/e1ffa6d4-4a72-4122-a6a7-e14b0a1fb984/wavy

Slimes chase you across an infinite tile map that spawns around the player. Waves get harder over time: spawn rate ramps every 15s, a second slime type shows up from wave 5, a boss every 15 waves. Killing slimes gives XP, which you spend on upgrades.

## Controls

- Move: WASD / arrows
- Attack: space or mouse
- Upgrade menu: E
- Upgrades: 1 damage, 2 speed, 3 health, 4 heal

## Running it

Open the folder in Unity Hub (2021.3 LTS). Press Play, or make a WebGL build.
