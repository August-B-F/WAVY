# Wavy

Wavy is a wave based survival game I made in a day for a school hackathon, in Unity.

![preview](Assets/preview.gif)

Slimes chase you across a tile map that spawns around you, you kill them for xp and spend it on upgrades, and it gets harder over time with a boss every 15 waves. You can play it in the browser, the link is in the repo. Move with wasd, attack with space or the mouse, and open the upgrade menu with e.
